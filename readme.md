**我想做一个AI驱动的无需代码的像空气一样随时可用的现代化数据处理平台**
# 是什么样的有什么价值?
现在数据分析要么使用spss,orange,需要回软件操作,要么使用r语言需要会编程.无论哪个都需要学习成本,这个学习成本不但包括原理编程的学习
甚至包括软件的安装,环境的配置,对于现在或者以后的移动互联网时代不怎么使用电脑的"Z时代"会是成本比较高的事情.所以我的想法是建立一个只用自然语言
就能实现科学数据处理甚至生产图像的平台,应该的形态是:得到数据后直接打字或者语音说"将某两组数据进行方差分析判断在0.001置信度内是否有差异"
然后得到结果,最好做成微信小程序以达到随时可用

方法块应当包括算法以及图像生成引擎,图像生成引擎应当能根据我们的自然语言需求对图像进行处理包括大小,颜色,样式等
AI能力目前应当采用deepseekR1可使用APIkey调用或者本地部署(可自行调教模型)
